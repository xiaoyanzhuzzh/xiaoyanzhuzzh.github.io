1. 一个页面从输入URL到页面加载完的过程中都发生了什么事情

  • DNS 域名解析
  • 建立 TCPIP 连接 —— 三次握手
  • 发起 HTTP GET 请求到服务器
  • 服务器处理请求并作出响应返回 HTTP Response
  • 浏览器拿到 Response 进行解析并渲染页面（DOM 解析, cssom 解析 (https://varvy.com/performance/cssom.html), js 解析等）
  • JS 解析这个过程中如果遇到需要请求外部资源 则跟上树一样发送http请求到相应浏览器

1.1 DNS 域名解析

  • DNS: Domain Name System - 域名系统
  • 因特⽹上作为域名和 IP 地址相互映射的⼀一个分布式数据库，能够使用户更方便的访问互联⽹。通过主机名，最终得到改主机名对应的IP地址的过程叫做域名解析。
