html之meta tag

最近在用HTML和CSS写一个个人的网页，关于meta的了解我知道的特别少，在这里写点笔记帮助自己理解一下，也希望给有需要的朋友们一些帮助。
首先，meta tag是写在网页的head tag中，title tag的下一行，不会显示在浏览器的视图窗口，在 HTML 中， 标签没有结束标签。
其次，meta tag的用法有很多，主要有两种属性：name和http-equiv。

大部份的META TAG设定与搜索引擎有关。

  • name属性主要描述网页，相对应的属性是content（网页内容），这些内容主要是便于搜索引擎机器人查找，分类。这其中最重要的是description（站点在搜索引擎上的描述）和keywords（分类关键字），所以应该给每页加一个meta tag.

    1. 自动更新

    设定合适的秒数，上例为每15秒重新下载（refresh）网页一次。很适合一些财经网站或运动网站，因为他们都提供【即时新闻】，每分钟自动更新网页一次，以展示当时的股票，指数或是比赛成绩等等。若希望网页每5分钟自动更新一次，就设定content=”300”.若您的网页内容是固定的，就没必要设定自动更新。即使设定了自动更新，网页内容也不会改变。

    2. 转址服务

    个人网页搬家时，较好的解决方法是保留原网页的首页，在上面等个乔迁声明及新址连接以引导网友，并可利用自动更新做为转址服务。通常设定3~10秒以展示一小段搬家讯息，您也可以设定为0秒，立即转址。

    3. 关键字

    许多网站提供了搜寻服务，meta tag就是用来协助搜索引擎，并且协助网友们找到您的网站，增加您的网站流量。
    在搜索网站的资料库中都要以一些关键字【keywords】作为网页索引，也要存一小段网页描述，所以可以使用meta tag直接告诉它。

    通常搜索引擎都会以网页的前15个字作为网页内容摘要，列于搜索结果上。但是有时候网页的前15个字只是一些不能代表网页的内容，例如：日期，时间等。由您提供网页的内容摘要为较好的方式。

    9.换页特效

    并不是每篇网页都需要换页特效或是适合使用换页特效，也不是四者一定要同时使用，可以自行决定使用时机。我建议使用在游记，相簿，或是介绍风景的网页上。
    http-equiv=”Page-Exit” 指定换页特效所发生的时机。

    http-equiv=”Page-Enter” 进入本页
    http-equiv=”Page-Exit”　离开本页
    http-equiv=”Site-Enter” 进入本站
    http-equiv=”Site-Exit” 离开本站

    content=”revealTrans(Transition=20)” 指定换页时的特效。

    特效有两种：blendTrans和revealTrans

    • content=”blendTrans(Duration=3.0)” 设定换页特效为【淡入淡出】。Duration=value是设定特效持续的时间（秒），过短过长都不好，太短看不出来，太长浪费浏览时间，建议在3-5秒，即 3.0 < value < 5.0。
    • content=”revealTrans(Transition=20)” 设定24种换页特效方式。Transition=value中的value可以是0-23。revealTrans可设定时间，如：content=”revealTrans(Transition=20 Duration=3.0)”

    10.换页特效

    默认是all
    其中的属性说明如下：

    设定为all：文件将被检索，且页面上的链接可以被查询；
    设定为none：文件将不被检索，且页面上的链接不可以被查询；
    设定为index：文件将被检索；
    设定为follow：页面上的链接可以被查询；
    设定为noindex：文件将不被检索，但页面上的链接可以被查询；
    设定为nofollow：文件将不被检索，页面上的链接可以被查询。
